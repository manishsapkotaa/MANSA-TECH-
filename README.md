## ğŸ¯ Customization

### Update Logo
Replace the logo URL in `index.html`:
```html
<img src="YOUR_LOGO_URL" alt="Mansa Technology">
```

### Update Contact Information
Edit contact details in the footer and contact section of `index.html`.

### Add/Edit Products
Modify the `products` array in `script.js`:
```javascript
const products = [
  {
    id: 1,
    name: 'Product Name',
    category: 'Category',
    price: 1000,
    // ... more fields
  }
];
```

### Add/Edit Services
Modify the `services` array in `script.js`:
```javascript
const services = [
  {
    id: 1,
    title: 'Service Name',
    description: 'Description',
    icon: 'iconName'
  }
];
```

### Change Colors
Update CSS variables in `styles.css`:
```css
:root {
    --color-primary: #2563eb;
    --color-primary-dark: #1d4ed8;
    /* ... more colors */
}
```
